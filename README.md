# ExamSystem

在线考试系统

分为三端：教师端、学生端、服务端

界面主要使用QWidget及其子控件实现，三端使用TCP/IP协议进行通信，传输的数据都封装为Json格式，并进行校验。

界面都实现为无边框窗口，适应扁平化显示风格。
